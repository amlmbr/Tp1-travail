Exercice 4 : Inscription de cours

Description:

Permet aux utilisateurs de s'inscrire à un cours et d'afficher un résumé.

Instructions
1.	Remplissez le formulaire.
2.	Appuyez sur le bouton pour afficher le résumé.

•	Java
